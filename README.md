# Advanced Vault

A time-locked ETH vault built in Solidity that allows users to securely deposit and withdraw funds after a defined lock period.

The goal of this project was to understand how to safely store funds in a smart contract and enforce time-based withdrawal conditions.

## Features

- Deposit ETH into the vault
- Set a lock duration for funds
- Withdraw ETH after unlock time
- View current vault balance
- Check unlock time
- Basic security checks for safe transfers

## Why I Built This

I built this contract to explore how vault-style contracts work and how time-based restrictions can be enforced on-chain.

This project helped me practice:

- handling ETH deposits
- managing timestamps
- writing safe withdrawal logic
- designing simple financial contract flows

## Tech Stack

- Solidity
- HTML
- CSS
- JavaScript (ethers.js)

## Network

Deployed on Sepolia testnet.

## Learning Focus

- time-lock mechanics
- ETH transfer safety
- smart contract state management
